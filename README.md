# Astra 2077 🚀
![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)

🔗 **Live Demo:** https://astra-2077.vercel.app/

A futuristic product showcase exploring how a single adaptive device can transform across multiple forms.

---

## 🧠 Concept

**Astra Zenith Flex** is a unified device that transitions seamlessly between:

- Smartphone  
- Wearable (Wrist Mode)  
- Tablet  
- Laptop  

The experience focuses on a **minimal, system-like interface** rather than a traditional webpage.

---

## ⚙️ Tech Stack

- Next.js  
- Tailwind CSS  
- React Three Fiber (Three.js)  
- GSAP (Scroll Animations)  

---

## ✨ Features

- Interactive 3D device  
- Scroll-driven transformation between modes  
- Subtle cursor-based interaction  
- Minimal futuristic UI
